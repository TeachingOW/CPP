3. Pseudocode for Checking Prime Number

Before writing the code, let's outline the logic using pseudocode:

Function isPrime(num):
    If num < 2:
        Return false
    If num == 2:
        Return true
    If num is even:
        Return false
    For i from 3 to sqrt(num), step by 2:
        If num % i == 0:
            Return false
    Return true

4. Optimizing the Check

To improve efficiency:

  • Skip even numbers after checking for 2.
  • Only check up to the square root of the number.
  • Use i += 2 to avoid unnecessary even checks.
